# Earth & Space Science - Unit 3 Assignment

## Viewing the website

If the shared website link does not work, you can still view the project on any computer:

1. Download or copy this entire project folder (keep the `images` folder with it).
2. Open `index.html` in a modern web browser, such as Chrome, Edge, Firefox, or Safari.

The page is self-contained and does not require installation, an account, or an internet connection. Double-clicking `index.html` should open it directly in your default browser.

> Tip: If it opens as text instead of a webpage, right-click `index.html`, choose **Open with**, and select a web browser.

For the most reliable result, do not move `index.html` away from `styles.css`, `script.js`, or the `images` folder, since the page uses those project files.
